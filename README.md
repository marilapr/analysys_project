# analysys_project
Проанализирована лояльность пользователей Яндекс Афиши. 
Наибольший потенциал удержания демонстрируют пользователи, начинающие с концертов и театров, совершающие покупки с десктопов и возвращающиеся с интервалом около трёх недель. Таким образом, лояльность клиентов напрямую зависит от регулярности покупок.
